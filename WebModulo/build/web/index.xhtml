Descripción Prime Faces

PrimeFaces es una biblioteca de componentes para JavaServer Faces de código abierto
que cuenta con un conjunto de componentes enriquecidos que facilitan la creación de
las aplicaciones web. Primefaces está bajo la licencia de Apache License V2.
rimefaces es uno de los frameworks web con mayor aceptación hoy día y que muchos
programadores web han puesto en práctica con resultados favorables, es una librería
de componentes visuales Open Source para JSF, así como IceFaces o RichFaces.
Según DevRates.com (Es un Sitio Web que mediante la opinión de los desarrolladores,
crean matrices de opinión acerca de todo lo concerniente a la programación) ,
Primefaces es el número dos en el top de los diez frameworks más populares del mundo.
